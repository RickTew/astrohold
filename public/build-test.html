SPEED CONTROL — VARIANT A: SEGMENTED THRUSTER BAR SANDBOX

Horizontal segmented bar. Each step is a chamfered tile that lights its own color (amber → cyan → red). Reads as a thruster gauge: more lit = more speed. Compact, slides into the HUD center panel's action row easily.

SPEED CONTROL — VARIANT B: TRIPLE CONSOLE BUTTONS SANDBOX

Three large chamfered push-buttons in a row, each with an icon glyph + label + cost-style mini caption. Mechanical translate-on-press, lit ring around the active one. Matches the existing tile-grid visual language so it can drop into any HUD panel.

SPEED CONTROL — VARIANT C: WARP-DIAL ROTARY SANDBOX

Circular instrument dial with three detents (8 / 10 / 12 o'clock). A glowing pointer rotates to the selected setting; an arc behind shows the range. Heaviest "spaceship cockpit" feel of the three; needs more vertical room than A or B.
